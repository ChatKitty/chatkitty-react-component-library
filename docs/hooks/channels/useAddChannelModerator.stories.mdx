# <u>useAddChannelModerator</u>

Note: This hook does not make an initial request on mount.

```typescript
const { resource, isLoading, error, makeRequest } = useAddChannelModerator();
```

## Input

| Parameter | Type      | Required | Defaults | Description               |
| :-------- | :-------- | :------- | :------- | :------------------------ |
| `client`  | ChatKitty | Yes      | n/a      | ChatKitty client instance |

## Output

| Parameter     | Type                                              | Description                                    |
| :------------ | :------------------------------------------------ | :--------------------------------------------- |
| `resource`    | Channel                                           | Channel that was updated                       |
| `isLoading `  | boolean                                           | Request loading state                          |
| `error`       | ChatKittyError                                    | If there's an error, it will be available here |
| `makeRequest` | `(channel: Channel, user: User) => Promise<void>` | Callback to request or re-request              |
