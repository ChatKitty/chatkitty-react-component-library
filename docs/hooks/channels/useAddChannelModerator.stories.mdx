# <u>useAddChannelModerator</u>

Note: This hook does not make an initial request on mount.

```typescript
const { resource, isLoading, error, makeRequest } = useAddChannelModerator();
```

## Input

| Parameter | Type      | Required | Defaults | Description               |
| :-------- | :-------- | :------- | :------- | :------------------------ |
| `client`  | ChatKitty | Yes      | n/a      | ChatKitty client instance |

## Output

| Parameter     | Type                                              | Description                                                       |
| :------------ | :------------------------------------------------ | :---------------------------------------------------------------- |
| `resource`    | Channel                                           | List of returned channels.                                        |
| `isLoading `  | boolean                                           | Function that can be called to fetch another page of channels.    |
| `error`       | ChatKittyError                                    | Total number of stored channels.                                  |
| `makeRequest` | `(channel: Channel, user: User) => Promise<void>` | If there's an error fetching channels, it will be available here. |
