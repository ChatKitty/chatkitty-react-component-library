# <u>useRemoveReaction</u>

Note: This hook does not make an initial request on mount.

```typescript
const { resource, isLoading, error, makeRequest } = useRemoveReaction();
```

## Input

| Parameter | Type      | Required | Defaults | Description               |
| :-------- | :-------- | :------- | :------- | :------------------------ |
| `client`  | ChatKitty | Yes      | n/a      | ChatKitty client instance |

## Output

| Parameter     | Type                                                 | Description                                    |
| :------------ | :--------------------------------------------------- | :--------------------------------------------- |
| `resource`    | Reaction                                             | Reaction that was removed                      |
| `isLoading `  | boolean                                              | Request loading state                          |
| `error`       | ChatKittyError                                       | If there's an error, it will be available here |
| `makeRequest` | `(message: Message, emoji: string) => Promise<void>` | Callback to request or re-request              |
