# <u>useDeleteUserBlockListItem</u>

Note: This hook does not make an initial request on mount.

```typescript
const { resource, isLoading, error, makeRequest } =
  useDeleteUserBlockListItem();
```

## Input

| Parameter | Type      | Required | Defaults | Description               |
| :-------- | :-------- | :------- | :------- | :------------------------ |
| `client`  | ChatKitty | Yes      | n/a      | ChatKitty client instance |

## Output

| Parameter     | Type                                         | Description                                    |
| :------------ | :------------------------------------------- | :--------------------------------------------- |
| `resource`    | User                                         | User that had block list updated               |
| `isLoading `  | boolean                                      | Request loading state                          |
| `error`       | ChatKittyError                               | If there's an error, it will be available here |
| `makeRequest` | `(item: UserBlockListItem) => Promise<void>` | Callback to request or re-request              |
